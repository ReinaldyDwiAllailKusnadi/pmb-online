# PMB Online

Portal Penerimaan Mahasiswa Baru (PMB) berbasis Laravel dengan UI Blade dan fitur pendaftaran, unggah dokumen, status, serta unduh bukti PDF.

## Stack

- Laravel 13
- Blade + Tailwind v4
- Vite

## Fitur Utama

- Autentikasi login/register
- Form pendaftaran bertahap (step 1-3)
- Upload dokumen dan status verifikasi
- Halaman status dan bukti pendaftaran PDF
